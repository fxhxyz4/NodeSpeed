features

        
Everything a terminal speedrunner needs


        

          

            
-l

            
Three languages

            
English, Russian and Ukrainian source texts, swappable with a single flag.

          


          

            
-c

            
Difficulty levels

            
Three complexity tiers, from warm-up paragraphs to dense technical text.

          


          

            
-m

            
Normal or timed

            
Race against a text with no clock, or set a hard time limit and beat it.

          


          

            
-s

            
Bring your own text

            
Point NodeSpeed at a local file or a URL and type that instead of the built-ins.

          


          

            
--stats

            
Local & global stats

            
Every attempt is logged locally, and synced to your account so you can track progress anywhere.

          


          

            
-o

            
Online duels

            
Quick-match a stranger or challenge a friend with a room code. Fastest, most accurate typist wins.

          

        

      


      

        
how duels work

        
One command, one code, one race


        
    
          
  1. 
            01
            
    
              
    Run nodespeed -o
    
              
    Choose quick match to find a random opponent, or create a private room for a friend.
    
            
    
          
    2. 
          
  2. 
            02
            
    
              
    Share the room code
    
              
    
                Creating a room hands you a short code like 7F3K9Q. Send it to
                whoever you want to race — they run nodespeed -o, pick "join", and paste it in.
              
    
            
    
          
    3. 
          
  3. 
            03
            
    
              
    Race, live
    
              
    
                Both terminals get the same text at the same moment. NodeSpeed compares accuracy first, then time, and
                announces a winner the second you both finish.
